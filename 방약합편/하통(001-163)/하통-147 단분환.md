### 구성
| 용량 | 약재명 |
| :--: | :--: |
| 0.4g | 경분 |
| 0.2g | 황단 석웅황 종유분  |
| 0.1g | 호박 유향 고백반  |
### 임상응용
治楊梅瘡  
양매창(매독)을 다스린다.  
  
<活套>  
  
1. 先傷腦後 [荊芥湯] 加藁本一錢  
   뇌후(뒤통수)에 먼저 생겼으면 형개탕에 고본 1돈을 가한다.
2.  頂前加 獨活  
   정수리에 생겼으면 독활을 가한다.
3.  足心加 桂皮  
   발바닥에 생겼으면 계피를 가한다.
4.  陰處加 靑皮  
   음부에 생겼으면 청피를 가한다.
5.  肢節加 穿山甲  
   팔다리의 뼈마디에 생겼으면 천산갑을 가한다.
6.  乳下加 柴胡  
   유방 밑에 생겼으면 시호를 가한다.
7.  口鼻加 枳殼  
   입과 코에 생겼으면 지각을 가한다.
8.  用藥後出喉症服 [ 仙遺粮湯] 三十貼  
   이 약을 쓴 뒤에 인후증이 생기면 [[하통-146 선유량탕]] 30첩을 쓴다.  
   
<用法> 
左末 나米糊和丸 梧子大 空心 荊芥雀舌煎湯呑下 十丸至十五丸  
위의 약을 가루내어 찹쌀풀에 반죽하여 오자크기로 환을 지어 공복에 형개 작설차 달인 물로 10~15환을 복용한다.
   
9. 선魚湯呑下 尤好 加麝香二分 不痛不痒不出喉症  
   선어(두렁이)탕으로 복용하면 더욱 좋다. 사향 2분(0.8g)을 더 넣으면 아프지도 않고 가렵지도 않고 인후증도 생기지 않는다.
