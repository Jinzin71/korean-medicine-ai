### 구성
| 용량 | 약재명 |
| :--: | :--: |
| 100g | 정향 목향 |
| 80g | 백초상(솥 밑에 그을음) 육두구 |
| 56g | 행인(거피첨) |
| 40g | 건강(포) |
| 2.8g | 파두 |
### 임상응용
治積痢 久痢 赤白膿血 及內傷生冷 藿亂 嘔吐  
적리와 오래된 이질, 적백리로 피고름을 누른 것과 날 것 또는 찬 음식에 상하여 곽란이 나서 토하는 증을 다스린다.  
  
<用法> 
先將 淸油一兩煎 蠟四兩熔化 次入右藥末 拌均 每一兩分作十丸 每一丸 米飮調服 或丸如菉豆 白湯下 十丸  
먼저 참기름 1냥을 끓인 데다 납 4냥을 녹인 다음 위의 약을 가루내어 넣고 반죽해서 매 1냥(40g)으로 30환을 지어 매회 1환을 미음에 타 먹거나 혹은 녹두 크기로 환을 지어 뜨거운 물로 10환씩 복용한다.

*행인(杏仁)은 껍질의 첨단을 잘라 버린다. 파두(巴豆)는 기름을 빼고 술에 삶는다.
