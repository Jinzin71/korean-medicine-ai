### 구성
| 용량 | 약재명 |
| :--: | :--: |
| 80g | 마아초 한수석 석고  |
| 40g | 감초 |
| 30g | 우담남성  |
| 6g | 자석영(수비) 우황 용뇌 사향 |
### 임상응용
治咽腫 口舌瘡 함頰腫熱 痰壅  
심화가 성하여 목안이 붓고 입안과 혀가 헐고 턱과 뺨이 부으면서 열이 나고 담이 막히는 증상을 다스린다.  
  
<用法> 左末 蜜丸 兩作三十丸 每一丸 薄荷湯 嚼下  
위의 약을 가루내 꿀에 개어 1냥(40g)으로 30환을 지어 매회 1환을 씹어서 박하 달인 물로 복용한다.

*자석영(紫石英)은 불에 달구어서 물로 깨끗이 씻어 쓴다.
