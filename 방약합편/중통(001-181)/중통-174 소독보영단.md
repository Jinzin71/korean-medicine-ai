### 구성
| 용량 | 약재명 |
| :--: | :--: |
| 60g | 전두등 |
| 40g | 산사 우방자 생지황 진사 |
| 28g | 적소두 승마 연교 |
| 20g | 형개 방풍 독활 감초 당귀 적작약 황련 길경 사과락 |
| 12g | 흑두 |

### 임상응용
每春秋分時 服一丸 痘毒漸消化  
매년 춘분과 추분때 1환씩 복용하면 천연두독이 점차 사라진다.  
  
<用法> 左藥末預辦 過春秋分 或上元七月望日 忌婦人猫犬 砂糖拌丸如李核 每服一丸 甘草湯化下  
위의 약을 가루내어 두었다가 춘분이나 추분을 지나 7월 15일쯤에 부인과 고양이 개를 피하고, 설탕에 반죽하여 배씨만하게 환을 지어 매회 1환을 감초탕에 녹여 복용한다.
